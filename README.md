# Sistema de Cadastro de Pessoas (Java)

Este projeto é um sistema simples de cadastro de pessoas desenvolvido em Java.

Ele permite:
- Cadastrar pessoas (nome e idade)
- Listar pessoas cadastradas
- Editar dados
- Remover pessoas
- Calcular a média das idades

O sistema funciona através de um menu no console.

## Tecnologias utilizadas

- Java 21
- IntelliJ IDEA
- Git e GitHub

## Como executar

1. Clone o repositório para sua máquina.
2. Abra o projeto no IntelliJ IDEA.
3. Execute a classe Main.java.
4. Utilize o menu no console para interagir com o sistema.
